# Vision-Conteo_Tornillos
Librería en Python para segmentar, contar y caracterizar objetos individuales presentes en una imagen cenital. 

Esta libreria tiene como objetivos

  • Mejorar la imagen mediante técnicas de preprocesamiento.  
  • Separar fondo y objetos mediante umbralización.  
  • Identificar regiones conectadas asociadas a cada objeto.  
  • Detectar bordes y características geométricas básicas.  
  • Estimar número total de objetos.  
  • Entregar la solución como paquete instalable y documentado.
  
Integrando temas como:

  • Thresholding  
  • Region-based segmentation  
  • Edge detection  
  • Corner detection  
  • Preprocesamiento
  
Incorporando procesos como:

  • Conversión de RGB a gris  
  • Filtrado espacial  
  • Al menos una transformación de intensidad  
  • Umbralización  
  • Segmentación de regiones  
  • Detección de bordes  
  • Medición de propiedades básicas de cada objeto  
  • Conteo final
